User Interface Design

This page presents some UI Designs made for the Mentorship System application. If you feel the UI Design can be improved and want to suggest something please discuss it with the community on AnitaB.org Open Source Zulip, we're always open to new ideas! Try to follow the Material Design guidelines when suggesting improvements.

Do you have an Android device? Here's the link to the latest version of the app: app-debug.apk
